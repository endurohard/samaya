# samaya — DIKIDI clone, design spec (6 модулей)

Дизайн-спецификация **готовая к реализации** в существующем стеке samaya: vanilla JS + чистый CSS, без фреймворков. Опирается на текущие компоненты (`subnav-item`, `btn-*`, `modal`, `row-item`, `data-card`, `cal-toggle`, `journal-period-btn`, `fin-kpi-card`, `bk-modal`) и переиспользует токены из `style.css` (`--card`, `--border`, `--primary`, `--success`, `--danger`, `--text-dim` и т.д.).

Цель — **DIKIDI feature-parity**, но с чистой современной вёрсткой; не повторять Bootstrap-3-вид DIKIDI 1:1.

## Общие принципы

### Layout shell
Все главные экраны живут внутри `.main`, имеют `.topbar` (бренд + тариф + помощь + bell + аватар), под ним — собственный тулбар экрана и контент. Сайдбар слева светлый (`--sb-bg #fff`).

### Subnav
Горизонтальный ряд `<button class="subnav-item">` с активным состоянием (`.active`) и опциональным `data-pending="P1"` бейджем для disabled-табов. Уже используется в Клиентах и Финансах.

```
[Расчёт зарплаты] [Взаиморасчёты] [Начисления] [Схемы расчёта]
       ^active
```

### Таблицы
Не `<table>`, а flex-row `.row-item` внутри `.data-list`. Шапка — отдельный `.data-list-head` с теми же grid-template-columns. Сортировка по клику на заголовок столбца (если применимо) — стрелка ▲/▼ справа.

### Модалки
`.modal-backdrop` (затемнение `rgba(15,23,42,0.5)`) + `.modal` (белая карточка). Структура: `.modal-head` (title + ×) → `.modal-body` (`form-grid` 2 колонки) → `.modal-foot` (primary справа, ghost слева). Esc и клик-вне закрывают. Ширина `min(540px, calc(100vw - 32px))` для обычных, `min(800px, …)` для крупных.

### KPI-карточки
`.fin-kpi-card` уже есть в Финансах: маленький LABEL (`text-transform:uppercase; font-size:11px; color:var(--text-dim)`) + большая цифра (`font-size:24px; font-weight:600`) + опциональный donut/иконка справа. Сетка `display:grid; grid-template-columns:repeat(N,1fr); gap:16px`. На мобилке — 1 колонка.

### Period pills
`<div class="period-pills">` с кнопками `<button data-period="day|tomorrow|week|month|year|custom">`. Активная — `.active` (фон `--primary`, текст белый). Уже работает в Журнале (`.journal-period-btn`) — переиспользовать класс или вынести в общий `.period-pill`.

### Иконки
Inline SVG 16–20px, `stroke="currentColor"`, `stroke-width="1.6"`, без icon-фонтов. Палитра — Heroicons outline.

### Цветовая семантика
| Назначение | Токен | Hex |
|---|---|---|
| Доход / положительный | `--success` | `#10b981` |
| Расход / отрицательный | `--danger` | `#ef4444` |
| Перемещение / нейтр. action | `--primary` | `#6366f1` |
| Мета / неактивное | `--text-muted` | `#9aa3b2` |
| Карточка | `--card` | `#fff` |
| Граница | `--border` | `#e3e8ee` |

### Состояния
- **Empty** — `<div class="empty">Транзакций пока нет. Нажмите «+ Доход».</div>` (центр, italic, `--text-dim`)
- **Loading** — skeleton-row с пульсацией (`@keyframes shimmer`)
- **Error** — pill `.error` (фон `--danger-soft`, рамка `#fecaca`, текст `#991b1b`)

### Адаптив
Desktop-first. На ширине ≤ 768px: subnav становится `overflow-x:auto`, модалки — full-screen, KPI-row — одна колонка, форм-сетка `1fr`.

### Без эмодзи в UI
Разрешены только: `⇄` (перемещение), `×` (закрыть), `⋯` (ещё), `↓` (загрузить), `+` (добавить), `▲/▼` (сортировка).
